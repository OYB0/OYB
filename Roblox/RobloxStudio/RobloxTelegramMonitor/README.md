# 📱 OYB Roblox & Telegram Integration System

## 🛠️ How to Setup & Install

To get this system working perfectly in your game, follow these steps:

1. **Create a Telegram Bot:** Open Telegram, search for `@BotFather`, create a new bot, and copy your **Bot Token**.
2. **Get your Chat ID:** Start a conversation with your bot to get your personal Chat ID.
3. **Insert the Script:** Place `TheScript.lua` inside **`ServerScriptService`** in your Roblox Studio Explorer.
4. **Configure the Settings:** Open the script and paste your `Bot Token` and `Chat ID` into the configuration variables at the top.
5. **Enable HTTP Requests (CRUCIAL):**
   * Open **Game Settings** in Roblox Studio.
   * Go to the **Security** tab.
   * Turn ON **"Allow HTTP Requests"** so the game can communicate with Telegram.
