## 👨‍💻 Overview
The official open-source repository for **OYB DEVELOPER**. Providing secure, optimized, and scalable systems for the Roblox ecosystem. All source code is built with a focus on performance and seamless integration.

## 🔭 Repository Directories
* 🔓 [**Roblox Executor Scripts**](https://github.com/OYB0/OYB/tree/main/Roblox/Executor): A directory containing a variety of custom scripts and utilities I developed for Roblox executors.
* 🛠️ [**Roblox Studio Systems**](https://github.com/OYB0/OYB/tree/main/Roblox/RobloxStudio): A collection of multiple systems and open-source resources crafted specifically for Roblox Studio projects.
* 📂 *(Check these directories to explore all the raw source code linked to my video tutorials).*

## 🛠️ Technical Stack
* **Languages:** Luau (Roblox), Python, JavaScript, HTML/CSS.
* **Specialties:** API Integration (Telegram/Discord), Secure Authentication, Backend Systems.

- 🔑 **[OYB Key System](https://github.com/OYB0/OYB/tree/main/Roblox/Executor/KeySystem):** Advanced authentication system.
- 📱 **[OYB Telegram Monitor](https://github.com/OYB0/OYB/tree/main/Roblox/RobloxStudio/RobloxTelegramMonitor):** Real-time Roblox-to-Telegram bridge.
